# PRD — 블로그 포스팅 도우미

## 서비스 개요

사용자가 네이버 스마트스토어 또는 네이버 지도 링크를 보면서 크롬 익스텐션을 실행하면, 자동으로 데이터를 수집하고 Claude AI가 블로그 포스팅 글을 생성한 뒤, 네이버 블로그에 글과 사진을 교차 삽입하며 자동으로 포스팅하는 서비스.

---

## 포스팅 종류 (전체 6개)

| 종류 | 데이터 소스 | 우선순위 |
|---|---|---|
| 쇼핑 | 네이버 스마트스토어 | 🔴 1순위 |
| 플레이스 | 네이버 지도 | 🟠 2순위 |
| 리뷰 | 미정 | 🟡 3순위 |
| 숙소 | 미정 | 🟡 4순위 |
| 여행지 | 미정 | 🟢 5순위 |
| 이슈 | 미정 | 🟢 6순위 |

---

## 시스템 구조

```
[크롬 익스텐션]
  ├── content script (스마트스토어)   → 상품 데이터 + 이미지 추출
  ├── content script (네이버 지도)    → 장소 데이터 추출
  ├── content script (네이버 블로그)  → 글자 입력 + 이미지 삽입 자동화
  └── popup UI                        → 진행 상태 표시

        ↕ HTTP

[Next.js 백엔드 (Vercel)]
  ├── POST /api/shopping              → 쇼핑 포스팅 생성
  ├── POST /api/place                 → 플레이스 포스팅 생성
  └── POST /api/images/save           → 이미지 다운로드 & Vercel Blob 저장
```

## 기능 상세

### 쇼핑 포스팅 (Sprint 2~4)

#### 데이터 수집 (content script — 스마트스토어)
- 상품명
- 가격 (정가 / 할인가)
- 상품 상세 설명
- 상품 이미지 URL 목록 (대표 이미지 + 상세 이미지)
- 리뷰 (평점, 주요 리뷰 텍스트, 리뷰 수)
- 배송 정보
- 판매자 정보

#### 포스팅 생성 (백엔드 — Claude)
- 수집된 데이터를 바탕으로 자연스러운 블로그 포스팅 글 생성
- 글 구조: 도입부 → 상품 소개 → 상세 정보 → 리뷰 요약 → 마무리
- 이미지 삽입 위치 지정 (글과 이미지 교차 구조)
- SEO 태그 생성

#### 이미지 처리
- 스마트스토어에서 추출한 이미지 URL → Vercel Blob에 다운로드 저장
- 저장된 이미지 URL 목록 반환

#### 블로그 자동 포스팅 (content script — 네이버 블로그)
- 네이버 블로그 에디터 페이지 진입
- 글자 한 자씩 타이핑 입력 (봇 감지 우회)
- 이미지 구간마다 저장된 사진 불러와서 삽입
- 글 → 사진 → 글 → 사진 순서로 교차 삽입
- 포스팅 저장

---

### 플레이스 포스팅 (Sprint 5~6)

#### 데이터 수집 (content script — 네이버 지도)
- 장소명
- 카테고리 (카페, 음식점 등)
- 주소 / 위치
- 영업시간
- 메뉴 및 가격
- 리뷰 (평점, 주요 리뷰 텍스트)
- 장소 이미지

#### 포스팅 생성 및 블로그 자동 포스팅
- 쇼핑과 동일한 흐름 (데이터만 다름)

---

## 전체 플로우

```
1. 사용자가 스마트스토어 or 네이버 지도 페이지 열기
2. 익스텐션 팝업에서 [포스팅 생성] 버튼 클릭
3. Content script가 현재 페이지에서 데이터 수집
4. 백엔드로 데이터 전송 → Claude가 포스팅 글 + 이미지 위치 생성
5. 이미지 Vercel Blob에 저장
6. 익스텐션이 네이버 블로그 에디터로 자동 이동
7. 글자 한 자씩 타이핑 + 이미지 교차 삽입
8. 저장 완료 → 팝업에 완료 표시
```

## 기술 결정사항

| 항목 | 결정 | 이유 |
|---|---|---|
| 크롤링 방식 | 크롬 익스텐션 content script | 서버 크롤링 시 네이버 차단, 비용 문제 |
| 이미지 저장 | Vercel Blob | Vercel과 통합, 무료 티어 충분 |
| 블로그 입력 방식 | 한 글자씩 타이핑 | 네이버 봇 감지 우회 |
| AI 모델 | claude-opus-4-6 | 최신 고성능 모델 |
| 배포 | 백엔드: Vercel / 익스텐션: Chrome Web Store | 각각 최적 플랫폼 |
